## How to use it?
1. clone the repo:
``` terminal
git clone https://github.com/spsaroj/craigs-list-clone.git
```
2. Go to the directory of the app
3. Run
``` terminal
python manage.py runserver
```
4. Enjoy searching in [browser](http://127.0.0.1:8000/)
